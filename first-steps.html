First steps

To get started with NixOS, you should just install and use the system for a bit before getting into more advanced configurations. To do that, you do not need any knowledge about how NixOS works, so just follow along.

Installing NixOS

This section is about the installation and first setup of NixOS.

TL;DR Ti's just like installing Linux Mint or Ubuntu.

After installing NixOS, you will find a file called configuration.nix under /etc/nixos. This is where we will configure our system.

Installing Packages

Installing packages on NixOS is different than on Debian-based systems and you have different options for how you want to install a package.

  1. nix-shell If you only need a small cli tool to run a few commands, you can install a package only for your shell session. This means that once you exit that shell, the program is removed again.
  2. system package

    System Package: environment.systemPackages = with pkgs; [ firefox ];

  3. user package:

    User Package: users.users.[username].packages = with pkgs; [ firefox ];


On NixOS, instead of running a command like sudo apt install firefox, you put a list of packages that you want into your configuration.nix and they will be installed for you. You can remove packages by simply deleting them from you list of packages.
You can install new packages on your NixOS system either as system packages or as user packages.
